MYBUDDY
MyBuddy is an on-the-go therapist! This solution is perfect for individuals who want to journal their mental-health progression while having someone to talk to; without feeling judged for sharing your emotions. MyBuddy uses Google Cloud's Speech-to-Text API for listening to your concerns to send off to OpenAI's API, (GPT-3.5 Turbo) for text-interpretation. With the usage of a unique prompt, the GPT-model is trained to "exercise patience over questions", getting rid of the AI language jargon; truly making MyBuddy communicate as your own friend!

STOCKEYE
StockEye is a discord bot developed with the intent to reduce a trader’s dependency on manual research efforts by automating the process of investment analysis. So far, on average, efficiently improved trader’s investment habits by 34% amongst a selection of manual versus bot-user participants. Holds the capability to create new channels for your investment portfolio, provide company research, provide live-market data and much more! Currently working on implementing a machine-learning algorithm to model/predict future stock movements from historical price data.
03
AUTOMAIL
AutoMail is a robust email automation script designed to simplify and streamline mass communication. Leveraging CSV data for personalized messaging, it effortlessly connects with a Gmail SMTP server to send tailored emails to a list of recipients. The script reads information from a CSV file and dynamically populates an email template for each recipient. The script demonstrates efficiency in managing email campaigns, making it an ideal tool for sending personalized messages to employees or registered users.
04
BLACKJACK
BlackJack is a Python script that simulates the classic card game Blackjack. The script employs object-oriented programming principles to model the game's various components. It includes functionalities such as deck management, player data storage, card display, a token system, and a comprehensive game history log. In addition to the core functionalities, the script also incorporates a robust game logic engine, allowing for the implementation of standard Blackjack rules and strategies.
